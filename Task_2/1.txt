for (auto&& x : v ) {...}
и
for (auto& x : v ) {...}

Будут отличаться, например, для контейнера vector<bool>.
В случае с (auto& x : v) x будет прокси объектом, который 
нельзя изменить - он является rvalue, поэтому данный код 
не скомпилируется.
Это связано с тем, что для хранения одного значения bool 
выделяется 1 бит.
